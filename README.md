# JavaScript Projects Collection

A collection of small JavaScript projects created to practice and strengthen core JavaScript concepts such as DOM manipulation, event handling, and interactive UI development.

These projects are built using **HTML, CSS, and Vanilla JavaScript** and focus on improving problem-solving skills and frontend development fundamentals.

---

## About This Repository

This repository contains multiple mini projects built with JavaScript.  
The goal of this repository is to practice real-world coding scenarios and improve my understanding of frontend development.

Each project demonstrates different JavaScript concepts including:

- DOM Manipulation
- Event Handling
- UI Interactions
- JavaScript Logic Building
- Basic Web App Functionality

---

## Tech Stack

Projects in this repository are built using:

- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
